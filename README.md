# QRcodeCreator
Alfred workflow to create QR code 

### 特性  
本地生成，快速预览
### 使用方法
下载 `QRcode Creator.alfredworkflow` 安装， `Alfred 3.1.1` 测试没有问题。
输入`qr` 后面跟上数据
